# a002. 簡易加法

## 題目說明

輸入兩個整數 `a`、`b`，請輸出它們的和。

## 輸入說明

每組測資只有一行，包含兩個整數 `a`、`b`，中間以空白隔開。

條件：

- `|a| < 10^6`
- `|b| < 10^6`

## 輸出說明

輸出 `a + b` 的結果。

## 範例

### 範例輸入 1

```text
5 10
```

### 範例輸出 1

```text
15
```

### 範例輸入 2

```text
1 2
```

### 範例輸出 2

```text
3
```

## 解題想法

這題只需要讀入一行資料，將輸入的兩個數字轉成整數後相加，最後輸出總和。

Python 可以用 `input().split()` 把一行輸入依照空白切開，再用 `map(int, ...)` 將每個字串轉成整數。

## 程式碼

```python
name = map(int, input().split())
print(sum(name))
```

## 程式碼說明

- `input()`：讀取一行輸入。
- `.split()`：依照空白把輸入切成兩個字串。
- `map(int, ...)`：把字串轉成整數。
- `sum(name)`：計算所有數字的總和。
- `print(...)`：輸出答案。
