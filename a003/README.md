# ZeroJudge a003 - 兩法師的占卜

這題會輸入兩個整數，分別代表月份 `M` 和日期 `D`。

依照題目公式計算：

```text
S = (M * 2 + D) % 3
```

再根據 `S` 的值輸出不同運勢：

- `S = 0` 輸出 `普通`
- `S = 1` 輸出 `吉`
- `S = 2` 輸出 `大吉`

## 解題思路

1. 讀入兩個整數 `M`、`D`
2. 計算 `(M * 2 + D) % 3`
3. 根據結果輸出對應的字串

## 輸入說明

輸入一行，包含兩個整數，分別為月份與日期。

## 輸出說明

輸出對應的運勢字串：

- `普通`
- `吉`
- `大吉`

## 範例

### 範例輸入 1

```text
1 1
```

### 範例輸出 1

```text
普通
```

### 範例輸入 2

```text
1 2
```

### 範例輸出 2

```text
吉
```

## 程式對應

`main.py` 使用 Python 讀入資料並直接依公式判斷輸出結果。
