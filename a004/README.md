# ZeroJudge a004 - 文文的求婚

這個專案是 ZeroJudge 題目 `a004` 的 Python 解法，用來判斷輸入的年份是否為閏年。

## 題目目標

程式會從標準輸入讀取多個年份，每讀到一個年份，就輸出該年份是：

- `閏年`
- `平年`

## 閏年判斷規則

一個年份符合以下任一條件時，就是閏年：

1. 可以被 `400` 整除。
2. 可以被 `4` 整除，而且不能被 `100` 整除。

否則就是平年。

## 程式說明

主要程式位於 `main.py`。

```python
if (year % 400 == 0) or (year % 4 == 0 and year % 100 != 0):
    print("閏年")
else:
    print("平年")
```

這段判斷式直接對應閏年的定義。

## 執行方式

在專案資料夾中執行：

```bash
python main.py
```

接著輸入年份，例如：

```text
2000
1900
2004
```

輸出結果：

```text
閏年
平年
閏年
```
